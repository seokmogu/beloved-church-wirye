## 2026-03-31 디자인 점검

### 🚨 긴급 이슈

#### 1. 네비게이션 메뉴 404 문제
**심각도:** 높음 (사용자 경험 차단)

네비게이션에 표시된 페이지들이 실제로 존재하지 않음:
- `/ministry` (예배) → 404
- `/sermons` (말씀) → 404  
- `/news` (소식) → 404
- `/bulletin` (주보) → 404

**현재 작동하는 페이지:**
- `/` (홈)
- `/about` (교회 소개)
- `/announcements` (공지사항)
- `/posts` (게시글)
- `/bulletins` (주보 목록)

**제안:**
- 네비게이션 메뉴를 실제 구현된 페이지와 동기화
- 또는 404 페이지를 대체할 임시 페이지 생성
- CMS에서 네비게이션을 관리할 수 있도록 구성

---

### 🎨 컬러 일관성

#### 2. 404 페이지 디자인
**현재 상태:**
- 단순한 텍스트와 버튼만 표시
- 브랜드 컬러(#1B3A2D, #C9A84C) 미사용
- 교회 정체성 부족

**개선안:**
- 브랜드 컬러를 활용한 일러스트레이션 추가
- 유도 문구 개선 ("페이지를 찾을 수 없습니다" → "찾으시는 페이지가 없거나 이동했습니다")
- 주요 페이지로의 빠른 링크 제공 (홈, 교회소개, 공지사항)

---

### 📱 레이아웃 & UX

#### 3. 메인 페이지 개선점

**히어로 섹션:**
- ✅ 브랜드 컬러 잘 활용됨
- ✅ CTA 버튼 배치 양호
- ⚠️ 모바일 반응형 확인 필요 (버튼 크기, 간격)

**공지사항 섹션:**
- ✅ 카드 디자인 깔끔
- 개선: 빈 상태 처리 (공지사항 없을 때 메시지)

**최신 설교 섹션:**
- ⚠️ 데이터가 없을 때 빈 공간만 표시됨
- 개선: skeleton loader 또는 "준비 중입니다" 메시지

**인스타그램 섹션:**
- ✅ 디자인 양호
- 개선: 로딩 상태 표시

**지도 섹션:**
- ✅ 정보 명확
- 개선: 지도 클릭 시 Google Maps/네이버 지도로 이동

#### 4. 교회 소개 페이지

**현재 상태:**
- 매우 간소한 텍스트 중심 레이아웃
- 브랜드 컬러가 거의 사용되지 않음
- 시각적 요소 부족

**개선안:**
- 섹션별 배경색 분리 (크림색 #F5F0E8 활용)
- 비전/우리이야기에 아이콘 또는 일러스트 추가
- 담임목사 프로필에 사진 추가 (CMS 필드)
- 타이포그래피 강화 (헤딩에 골드 악센트)
