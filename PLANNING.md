# PLANNING.md - 사랑하는교회 위례 웹사이트 개선 계획

## 2026년 3월 31일 기획 분석

### 현재 상태 파악
- **메인 페이지**: 깔끔한 디자인, 예배 안내, 공지사항, 설교, 인스타그램 피드, 지도 등 핵심 정보 포함
- **네비게이션**: 교회 소개, 예배, 말씀, 소식, 주보, 검색 메뉴 구성
- **소셜 미디어**: YouTube, Instagram 연동
- **기술 스택**: Next.js + Payload CMS, 반응형 디자인

### 개선 제안 (우선순위 순)

#### 1. **새가족 등록 시스템 구축** (높음)
- **현황**: 현재 새가족 등록을 위한 전용 페이지/폼이 없음
- **제안**: 
  - 새가족 온라인 등록 폼 페이지 추가 (`/newcomers` 또는 `/register`)
  - 입력 항목: 이름, 연락처, 이메일, 주소, 처음 방문/등록 의사, 간단한 신앙 배경
  - 등록 후 자동 환영 메시지 발송 및 담당 교역자 알림
  - 메인 페이지 상단 CTA 버튼 추가 ("새가족 등록" 또는 "처음 오셨나요?")
- **참고**: 소망교회, 혜린교회 등 대형 교회들은 새가족 등록 프로세스를 웹사이트에 명확히 안내

#### 2. **헌금 안내 및 계좌 정보 개선** (높음)
- **현황**: 헌금 안내가 Footer에만 간략히 언급되거나 누락되어 있을 가능성
- **제안**:
  - 헌금 안내 페이지 추가 (`/giving` 또는 `/offering`)
  - 계좌 정보, QR코드(카카오페이/토스 등), 자동이체 안내 제공
  - 헌금 항목별 구분 (주일헌금, 감사헌금, 선교헌금, 건축헌금 등)
  - Footer에 "헌금 안내" 링크 추가
- **벤치마킹**: QR코드를 활용한 간편 헌금 시스템 도입 (네이버 블로그 사례 참고)

#### 3. **소그룹/사역 안내 페이지 확장** (중)
- **현황**: "교회 소개" 페이지에 간략한 설명만 있음
- **제안**:
  - 소그룹 목록 및 소개 페이지 추가 (`/groups` 또는 `/ministries`)
  - 각 소그룹별 요일, 시간, 장소, 참여 방법 안내
  - 소그룹 참여 신청 폼 또는 연락처 제공
  - 사역 팀 소개 (예배팀, 미디어팀, 교육팀 등)
- **효과**: 교인들의 공동체 참여도 증가, 새가족이 쉽게 소그룹 찾기 가능

#### 4. **웹 접근성 개선** (중)
- **현황**: 기본적인 시맨틱 HTML 구조는 있으나 접근성 도구 테스트 필요
- **제안**:
  - WAVE 또는 Lighthouse 접근성 테스트 수행
  - 이미지 alt 텍스트 보완 (특히 인스타그램 임베드 및 YouTube 썸네일)
  - 키보드 네비게이션 및 스크린리더 최적화
  - 색상 대비 검토 (WCAG AA 준수)
  - ARIA 레이블 추가 (모달, 내비게이션 메뉴 등)
- **참고**: ResourceUMC 접근성 가이드 참고

#### 5. **SEO 최적화 강화** (중)
- **현황**: 기본 메타태그는 있으나 구조화된 데이터 누락 가능성
- **제안**:
  - Schema.org 구조화 데이터 추가 (Church, Event, Organization)
  - 각 페이지별 고유한 메타 디스크립션 및 Open Graph 태그
  - 사이트맵(sitemap.xml) 생성 및 Google Search Console 등록
  - 지역 SEO 최적화 ("위례 교회", "남위례역 교회" 등 키워드)
  - 블로그/설교 콘텐츠에 의미 있는 제목 및 요약 추가
- **효과**: 지역 검색 시 노출도 증가, 신규 방문자 유입
